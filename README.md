# Weather App

A responsive Weather App built using React and Vite.

## Features

- Search weather by city
- Real-time weather data
- Temperature
- Humidity
- Wind Speed
- Weather Icons
- Responsive Design
- Fetch API with Async/Await

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- OpenWeather API

## Installation

```bash
npm install
npm run dev
```
